## 요청개요
Veo 3.1로 어린이 단어학습 영상을 만들기 위한 세부 구체적인 script를 작성해줘

각 영상만 보아도 학습단어가 생각이 날수 있는 연출의 prompt작성

아래에 나열된 전체영상 학습단어의 갯수만큼 prompt를 별도로 작성해줘
나한테 설명하는 부분과 별도로, 각 Scene 마다 복사해서 사용할수 있도록 구분 작성해주고
각 prompt는 독립적으로 필요한 부분이 각각 포함되도록 작성해줘
각 prompt에는 image to video를 사용해서 일관성을 유지하라는 말을 넣어줘
각 프롬프트에 나레이션이 일괄성을 가지도록 구체적 톤과 방식을 지정(케릭터별, 나레이터)
timeline별로 디테일한 상황설명을 포함한 prompt를 작성해줘
각 프롬프트에 일관성을 위한 공통부분(목소리, 스타일)을 반복해서 항상 표시해줘

## 세부 요구사항

어린이들의 흥미를 끌어내기 위해 위해 웃기게 연출 slapstick cartoon 되어야 함
톰과 제리같이 웃긴 장면들의 연출이 요구됨
각 장면들과 단어들의 연결을 통해 강력한 anchoring으로 단어 학습을 효과적으로 되어야 함
제공된 케릭터들 이미지를 활용해서 만들 것 (강아지 Max , 아이 Ido)
정확한 철자를 위해 프롬프트 마지막에 with clear text "WORD" displayed on the top right 식으로 위치를 지정해줘
글자는 아주 잘보이게 일괄성있게 참조이미지 hick, bold, Play-Doh style clay animation text 스타일로보이게 해줘
각 prompt에 학습할단어를 포함한, 상황에 맞는 적절한 대사(영어)나 narration을 포함

각 장면을 생성할 때 일관성을 잘 유지하기 위한 반복적인 스타일 및 목소리 prompt를 맨앞에 고정으로 넣어줘
예를들어 - Slapstick cartoon style, 3D render, vibrant colors, Max the dog and Ido the kid, funny expression 같은 키워드를  넣어 일관성을 유지해줘
